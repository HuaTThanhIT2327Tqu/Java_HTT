#  Java_HTT – Quản Lý Bán Hàng

##  Giới thiệu
Dự án **Quản Lý Bán Hàng** được xây dựng nhằm hỗ trợ việc quản lý sản phẩm, hóa đơn và khách hàng trong cửa hàng bán lẻ.  
Ứng dụng được phát triển bằng **Java** và kết hợp giao diện đồ họa (Java Swing/JavaFX – tùy theo cách bạn code), giúp người dùng thao tác trực quan và dễ sử dụng.

Mục tiêu chính của dự án:
- Hỗ trợ quản lý thông tin sản phẩm
- Xử lý hóa đơn bán hàng
- Theo dõi khách hàng
- Lưu trữ và truy xuất dữ liệu nhanh chóng
- Tạo môi trường thực hành lập trình Java cho sinh viên


## Cách chạy ứng dụng

### Nếu chạy bằng file JAR:

1. Mở terminal/cmd.
2. Chạy lệnh: duong_dan_den_file\QLBH_HuaTheThanh_DHCNTTK4.jar

## Nếu chạy bằng mã nguồn:
- Mở project trong **IntelliJ IDEA / NetBeans / Eclipse**.
- Build project và chạy file `QLBH_Main.java`.
